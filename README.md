# StudyFlow AI

Study Smarter, Not Harder — an AI-powered study planner that helps students organize tasks, focus deeper, and track progress with a modern, glassmorphic UI.

🔗 **Live demo:** https://genesis-studyflow.lovable.app

## ✨ Features

- **Landing page** — hero, productivity stats, and feature showcase
- **Dashboard** — weekly focus charts, goal progress, subject distribution, and task management
- **AI Planner** — smart, priority-based daily schedule generation with recommendations
- **Focus Timer** — Pomodoro timer with session tracking and break reminders
- **Modern UI** — dark mode, glassmorphism, purple/blue gradients, fully responsive

## 🛠 Tech Stack

- **Framework:** React 19 + TanStack Start (Vite 7)
- **Routing:** TanStack Router (file-based)
- **Styling:** Tailwind CSS v4 + custom design tokens (oklch)
- **UI:** shadcn/ui + Radix primitives
- **Charts:** Recharts
- **Icons:** Lucide React
- **Language:** TypeScript
